# Định hướng dự án

Đây là file định hướng dự án. Hãy sử dụng n8n MCP Server và n8n Skills để giúp tôi tự động xây dựng các workflow.

## Mục tiêu

Dự án này tập trung vào việc tự động hóa việc xây dựng và quản lý các workflow trong n8n thông qua:
- n8n MCP Server: Kết nối và tương tác với n8n instance
- n8n Skills: Sử dụng các kỹ năng chuyên biệt để tạo và quản lý workflow

## Nguyên tắc làm việc

- Ưu tiên sử dụng n8n MCP Server để thực hiện các tác vụ liên quan đến n8n
- Tận dụng n8n Skills để tự động hóa quy trình xây dựng workflow
- Đảm bảo workflow được xây dựng theo các best practices của n8n
